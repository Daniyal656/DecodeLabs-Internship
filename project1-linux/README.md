# Project 1 - Linux & command Line Basics 
## Goal
Learn and use basic Linux commands required in DevOps environments.


## Commands Used
- mkdir -p : Create nested directories
- touch : Create empty files
- echo : Write content to files
- ls -R : List directory structure recursively
- mv : Move/rename files
- ls -l : View file permissions
